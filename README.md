# 🖥️ OS Playground

**Interactive visualization platform for Operating System algorithms and concepts.**

OS Playground helps students and developers understand complex Operating System concepts through beautiful interactive simulations, animations, and real-time algorithm execution.

---

## 🚀 Features

### CPU Scheduling Visualizer
Visualize popular CPU scheduling algorithms:
- FCFS
- SJF
- SRTF
- Priority Scheduling
- Round Robin
- Multilevel Queue
- Multilevel Feedback Queue

Features:
- Interactive process input
- Gantt chart visualization
- Waiting time calculation
- Turnaround time analysis
- Response time tracking

---

### Memory Management Visualizer
Explore memory allocation strategies:
- First Fit
- Best Fit
- Worst Fit
- Next Fit
- Buddy System

Features:
- Dynamic memory block simulation
- Fragmentation analysis
- Allocation/deallocation visualization

---

### Page Replacement Visualizer
Understand page replacement concepts:
- FIFO
- LRU
- Optimal
- Clock Algorithm
- LFU

Features:
- Page hit/miss tracking
- Memory frame animation
- Hit ratio comparison

---

### Disk Scheduling Visualizer
Visualize disk head movement algorithms:
- FCFS
- SSTF
- SCAN
- C-SCAN
- LOOK
- C-LOOK

Features:
- Disk movement animation
- Seek time calculation
- Performance comparison

---

### Deadlock Visualizer
Learn deadlock prevention and detection:
- Banker's Algorithm
- Resource Allocation Graph
- Deadlock Detection

Features:
- Safe sequence simulation
- Process-resource mapping
- Interactive graph visualization

---

### Process Synchronization
Simulate synchronization problems:
- Producer Consumer
- Dining Philosophers
- Readers Writers
- Sleeping Barber

Features:
- Thread execution visualization
- Semaphore and mutex behavior
- Queue state animations
